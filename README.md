# Inventory API 
-
Este proyecto fue realizado con el objetivo de aplicar conocimientos aprendidos en la cátedra Arquitecturas Web, usando Spring Boot como framework y arquitectura monolítica.
---
## Tecnologías
- Spring Boot
- Spring Data JPA
- Spring Security
- PostgreSQL Driver
- Docker
---
## Funcionalidades
La API se basa en un inventario de almacén, buscando optimizar el almacenamiento de artículos del mismo y llevar una trazabilidad, permitiendo:
- CRUD de artículos
- CRUD de grupos de artículos
- Clasificación de artículos en grupos
- Registro de movimientos de entrada y salida de artículos del inventario con actualización automática de stock para mantener coherencia
- Roled Based access dejando ciertas funcionalidades aptas solo para administradores
